# Off-Site Authority Checklist

Status: Active
Last updated: 2026-06-17
Owner: Agent
Source of truth: approved public proof, current profiles, HydrogenExpert pages

## Purpose

Track truthful off-site authority opportunities without inventing Shopify Partner status, Shopify Plus Partner status, office locations, agency team size, client metrics, ratings, testimonials, or directory eligibility.

## Can Maintain Now

- LinkedIn profile: keep Hydrogen, Liquid, Shopify, senior implementation, and direct operator positioning aligned with the site.
- Upwork profile: keep public proof and service wording aligned with fixed-scope Hydrogen, Liquid/no-rebuild honesty, and approved case evidence.
- Udemy or teaching profile: use only existing public teaching facts and do not imply Shopify endorsement.
- Personal portfolio links: keep HydrogenExpert, case studies, package pages, and scope review routes consistent.
- GitHub profile or repos: link only if repositories are public, relevant, and safe to show as code or tooling proof.

## Requires Proof or Access First

- Shopify Partner Directory or category listings: only claim if eligibility and public listing are real.
- Shopify Plus Partner language: do not use unless the status is official and publicly verifiable.
- Agency awards, badges, client logos, ratings, or certifications: use only when approved and public.
- US or UK local-office references: do not use; remote collaboration is the truthful positioning.

## Quarterly Review

- Confirm profile links still work.
- Confirm profile claims match current HydrogenExpert pages.
- Remove stale service claims or unsupported proof.
- Add new public proof only after it is approved.
