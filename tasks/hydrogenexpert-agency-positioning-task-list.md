# HydrogenExpert.co Agency Positioning Task List

Audit date: 2026-05-05
Scope: HydrogenExpert.co only
Primary question: Can HydrogenExpert.co serve as a US/global Hydrogen Shopify agency asset?

## Evidence Snapshot

- Live homepage H1: "Shopify Hydrogen developer for growing brands"
- Live homepage positioning: "without agency noise", "direct operator", "no agency handoffs", "independent Shopify Hydrogen developer"
- Live services H1: "Shopify Hydrogen services without the agency maze"
- Existing commercial pages include agency alternative, US agency alternative, headless agency alternative, developer, audit, migration, SEO, cost, custom storefront, performance optimization, and support retainer pages.
- Live sitemap contains 35 URLs, including 10 service/commercial pages, 3 case studies, 4 blog posts, `/blog`, `/about`, `/contact`, and core decision pages.
- Live robots.txt allows all crawlers and points to the sitemap.
- Live llms.txt exists and describes the asset as "Shopify Hydrogen Developer", not as an agency.
- Shopify documentation grounding checked through the installed `shopify-dev` skill. Relevant official references: Hydrogen is Shopify's React Router based custom storefront toolkit optimized for Storefront API data, Oxygen is Shopify's edge hosting layer for Hydrogen, and Hydrogen/Headless channels manage storefront tokens and deployment workflows.

## Strategic Decision

HydrogenExpert.co can be used as a US/global Hydrogen Shopify services asset, but not as a classic "agency" asset in its current form.

Current perception: senior independent Hydrogen expert / consultant with strong proof.

Recommended positioning: "senior-led Hydrogen Shopify studio" or "agency alternative" rather than pretending to be a multi-person US agency. The existing domain and proof are useful, but the site needs clearer offer packaging, stronger buyer-path CTAs, service-page intent separation, and more source-grounded commercial content before it can reliably compete for "Hydrogen Shopify agency" and "headless Shopify agency" intent.

Separate US agency site: not necessary for the first 30 days. First, make HydrogenExpert.co carry the agency-intent pages honestly. A separate US agency domain only becomes useful if Emre wants to sell a separate team/company narrative with US-local proof, multi-role delivery, or broader Shopify services beyond Hydrogen.

## First 30-Day Priority Order

1. Reframe homepage and llms.txt from "developer only" to "senior-led Hydrogen Shopify studio / agency alternative".
2. Strengthen the top 6 commercial service pages with clear offer structure, deliverables, process, qualification, and CTAs.
3. Add a primary conversion path around "Request Hydrogen Fit Audit" and "Book a Scoping Call".
4. Build a keyword map and internal linking system from blog and case studies to service pages.
5. Run Shopify-doc-grounded technical content QA on all Hydrogen/Oxygen/Storefront API claims.
